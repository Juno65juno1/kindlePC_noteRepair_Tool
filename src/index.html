你好, 我是 KindleNT :-)
一款处理 KindlePC 笔记的在线工具

产品说明

此工具专为那些希望将「KindlePC 导出的 HTML 笔记」转化为「更便于阅读与管理的美观 HTML 文件」的用户而设计。

通过精细的转换流程，它保证了笔记内容的整洁性与易读性，使得管理和回顾变得轻松愉快。

使用方法

第一步：点击页面上的按钮。

第二步：在弹出的文件选择对话框中，选择需要处理的文件。

第三步：文件上传后，系统将自动开始处理。请耐心等待处理完成。

第四步：处理完成后，弹出对话框提示下载。选择保存位置并下载处理后的文件。

注意事项

1. 确保上传的文件是「Kindle 电脑端」导出的文件，并且其后缀为 .html。

2. 处理过程中，请勿关闭页面或关闭浏览器，以免影响转换效果。
